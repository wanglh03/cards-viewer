# 香港开户指南

## 汇丰香港

服务费用：https://www.hsbc.com.hk/zh-cn/fees/

### 申请PLUS提款卡

银联提款卡对应着港币储蓄账户（883结尾），PLUS提款卡对应港币往来账户（001结尾）。

PLUS提款卡在全球ATM提款手续费为40HKD，银联提款卡则为50HKD。

填写以下表格：

- [PLUS自動櫃員機晶片卡登記表格(綜合理財戶口)](https://www.hsbc.com.hk/content/dam/hsbc/hk/docs/accounts/personal-integrated/plus-atm-chip-card-enrolment-form.pdf)

可以前往分行或者寄信申请。寄信申请可通过邮寄方式将卡片寄回您的通讯地址，后续会有汇丰的工作人员电话确认，并将卡片和密码函寄出。

地址为：九龙邮政总局72677号信箱，PO Box 72677，Kowloon Central Post Office，Kowloon，Hong Kong。

后续到香港以后可以通过信件上的单号在 [香港邮政](https://webapp.hongkongpost.hk/sc/mail_tracking2) 追查。

参考文献：[汇丰香港plus提款卡之申请](https://post.cplus8.com/d/832)
