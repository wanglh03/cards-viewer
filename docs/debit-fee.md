# 各行借记卡费用一览

## 前言

国家发展改革委 中国银监会[《关于取消和暂停商业银行部分基础金融服务收费的通知》](https://www.ndrc.gov.cn/xxgk/zcfb/ghxwj/201707/t20170710_960917.html)(发改价格规〔2017〕1250号)规定，各行可指定1张借记卡免收年费和小额账户管理费。

借记卡的费用主要包括六种：开卡工本费、年费、小额账户管理费、动账短信通知费、跨行转账手续费、境内跨行ATM取现手续费。

以下讨论的“双免”指免除年费、小额账户管理费。以下讨论的范围为“二卡”，即指第2-4张（或更多）借记卡。

二卡如需双免，可以办理各行“乡村振兴”卡、“新市民”卡、社保卡等卡种。邮储U+卡可六免。

## 21家国内系统重要性银行

所有等级卡双免：中行、邮储、中信、兴业、民生、浦发、光大、华夏

普卡双免：平安、招商

只免年费：交通、广发

招商金葵花卡可以代发或特定城市减免。
